1. Differentiate between Harvard and Von-Neumann Architecture The Harvard architecture is a computer architecture with separate storage and signal pathways for instructions and data. It contrasts with the von Neumann architecture, where program instructions and data share the same memory and pathways.
2. What is instruction? An instruction is an order given to a computer processor by a computer program. 4. Write short notes on:
(a) Cache memory Cache memory is a chip-based computer component that makes retrieving data from the computer's memory more efficient. It acts as a temporary storage area memory.
(b) RAM Random Access Memory This is the type of memory that stores information for a short period of time (Temporarily). It also holds instructions for the computer, its programs and data. It can be read and written, volatile. It makes the computer run faster if it’s installed in bigger volumes for example 10GB of RAM.
(c) ROM Read Only Memory This is the type of memory that stores information for a long period of time (Permanently). It is built memory of a computer. It is nonvolatile memory because it never loses its contents. This type of memory lets you store the data needed to start up or boot the computer.
(d) Interrupt These are signals sent to the Central Processing Unit by external devices, normally input and output devices. They tell the Central Processing Unit to stop its current activities and execute the appropriate part of the operating system. Hardware Interrupts are generated by hardware devices to signal that they need some attention from the Operation System.
